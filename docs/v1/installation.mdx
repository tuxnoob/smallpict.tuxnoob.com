# Configuration & Setup Guide

This document outlines the installation, configuration, and feature tiers for the SmallPict WordPress plugin.

# Installation & Setup

Getting started with SmallPict is incredibly simple. You can be up and running in less than 2 minutes.

## 1. Installation

### From WordPress Dashboard
1.  Go to **Plugins > Add New**.
2.  Search for **SmallPict**.
3.  Click **Install Now**, then **Activate**.

### Manual Installation
1.  Download the latest `.zip` file from [TuxNoob.com](https://tuxnoob.com) or the WordPress repository.
2.  Go to **Plugins > Add New > Upload Plugin**.
3.  Select the zip file and click **Install Now**.

---

## 2. Activation & Plan Selection

Upon activation, you will be greeted by the setup wizard (powered by Freemius).

1.  **Opt-in**: Click "Allow & Continue" to enable updates and activate the Free Tier.
2.  **Upgrade (Optional)**: If you need **ultra-light image formats**, **automatic resizing**, or **more quota**, you can upgrade to a Pro plan directly from the dashboard.

<Callout type="tip">
**That's it!** No API keys, no cloud servers to manage, no complex configuration. SmallPict is now protecting your site speed.
</Callout>

---

## 3. Configuration

Go to **Settings > SmallPict** to customize your preferences:

*   **Auto Optimize**: Ensure this is checked to optimize images as soon as you upload them.
*   **Compression Mode**:
    *   **Lossy (Recommended)**: Best balance of size and quality.
    *   **Lossless**: Preserves 100% of pixel data (Pro only).
*   **Output Format**: Choose **WebP** for broad compatibility or **AVIF** (Pro) for cutting-edge compression.
