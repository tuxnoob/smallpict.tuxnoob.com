## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- An active internet connection (for cloud processing)

## Install from WordPress.org

1. Go to **Plugins → Add New** in your WordPress admin
2. Search for **"SmallPict"**
3. Click **Install Now**, then **Activate**

## Manual Installation

1. Download the plugin ZIP from [WordPress.org](https://wordpress.org/plugins/smallpict/)
2. Go to **Plugins → Add New → Upload Plugin**
3. Choose the ZIP file and click **Install Now**
4. Click **Activate Plugin**

## After Activation

Once activated, SmallPict will redirect you to its setup page where you can:

1. **Enter your email** to create a free account
2. **Verify with OTP** — a 6-digit code sent to your email
3. **Start optimizing** — your API key is saved automatically

No copy-pasting required. The entire setup takes under 60 seconds.
