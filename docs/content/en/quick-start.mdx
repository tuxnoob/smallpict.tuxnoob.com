## Your First Optimized Image

After completing the [Installation](/docs/installation), SmallPict is ready to work.

### Step 1: Upload an Image

Go to **Media → Add New** and upload any image (JPEG, PNG, or GIF).

### Step 2: Automatic Processing

If **Automatic Optimization** is enabled (it is by default), SmallPict will:

1. Upload your image to our secure cloud pipeline
2. Compress it using lossy compression at quality 80
3. Convert it to **WebP** format
4. Replace the original with the optimized version
5. Update all WordPress thumbnail sizes

### Step 3: Check the Results

Go to **Media → Library** and click on the image. You'll see:

- **Original size** vs **optimized size**
- **Savings percentage**
- **Output format** (WebP)

### Step 4: Monitor Your Quota

Visit the **SmallPict** menu in your WordPress admin sidebar. The sidebar shows:

- **Monthly quota usage** as a progress bar
- **Used / Remaining / Limit** in MB
- **Current plan** tier

## Default Settings

| Setting | Default Value |
|---------|---------------|
| Auto Optimize | ✅ Enabled |
| Compression Mode | Lossy |
| Output Format | WebP |
| Quality | 80 |

You can change these anytime from the SmallPict settings page. See [Image Settings](/docs/image-settings) for details.
