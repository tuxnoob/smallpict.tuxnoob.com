## What is SmallPict?

SmallPict is a **cloud-powered image optimization service** built specifically for WordPress. It compresses, converts, and resizes your images using serverless infrastructure — so your site stays fast without overloading your server.

## How It Works

1. **Install the plugin** from WordPress.org
2. **Verify your email** using a one-time code (OTP)
3. **Upload images as usual** — SmallPict optimizes them automatically

Every image uploaded to your WordPress Media Library is sent to our secure cloud pipeline, compressed using industry-leading algorithms, and returned in modern formats like **WebP**.

## Key Features

- **Automatic Optimization** — No manual work required after setup
- **WebP Conversion** — Modern format with superior compression
- **Quality Control** — Adjustable quality slider (50–95)
- **Quota Dashboard** — Real-time usage tracking inside WordPress
- **Privacy-First** — Your images are processed and immediately deleted from our servers
